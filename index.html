OpenFV is now live! Github repository here.

OpenFV is the first comprehensive open source library focusing at 3D Particle Image/Tracking Velocimetry (PIV/PTV). As of now, OpenFV primarily supports:

  • Synthetic Aperture (SA) Reconstruction
  • Tomographic Reconstruction
  • 3D PIV (window deformation supported)
  • Relaxation based 3D Particle Tracking
  • Synthetic Particle and Velocity Field Simulation
  • All the features mentioned above are GPU and CPU parallelized to varying degrees.
